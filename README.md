# PatternExplorer

### Clone the repository:

    git clone https://github.com/rahulrana701/PatternExplorer.git

### Start the app:

     cd PatternExplorer
     cp .env.example .env
     npm install
     npm run dev
     
Access the application:

  Open http://localhost:3000 in your browser.
